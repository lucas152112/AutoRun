# Examples

This directory contains example scripts demonstrating how to use AutoRun.

## Available Examples

### basic_example.py
Demonstrates basic task creation and scheduling:
```bash
python examples/basic_example.py
```

### config_example.py
Shows how to work with configuration:
```bash
python examples/config_example.py
```

## Running Examples

To run the examples, make sure you have installed AutoRun:

```bash
# Install in development mode
pip install -e .

# Run an example
python examples/basic_example.py
```
