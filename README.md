# AutoRun - 自動啟動排程工具

Windows 內建的排程功能不夠好用，所以重新開發了一個更簡單、更直觀的排程管理工具。

## ✨ 功能特色

### 核心功能
- 📅 **排程管理**：設定程式在指定時間和星期自動執行
- ⏰ **分鐘精度**：支援精確到分鐘的排程設定
- 🔄 **週期選擇**：可選擇每天或指定星期（一到日）
- ▶️ **一鍵啟動**：簡單的排程器開關控制
- 📝 **執行紀錄**：即時顯示排程執行狀態和結果

### 系統整合
- 🖥️ **系統托盤**：最小化到托盤，不佔用工作列空間
- 🔔 **氣泡通知**：排程執行時顯示提醒通知
- 🚀 **開機啟動**：支援 Windows 開機時自動啟動
- 💾 **自動儲存**：排程設定自動保存，不怕遺失

### 使用者介面
- 🎨 **清晰界面**：直觀的排程列表和編輯介面
- 🔒 **智能狀態**：未編輯時自動禁用輸入欄位
- ⚡ **快速操作**：新增、編輯、刪除排程只需幾秒
- 📊 **即時更新**：排程狀態和執行紀錄即時顯示

## 🚀 快速開始

### 系統需求
- Windows 10/11
- .NET 8.0 Runtime

### 下載安裝
1. 從 [Releases](https://github.com/lucas152112/AutoRun/releases) 下載最新版本
2. 解壓縮到任意資料夾
3. 執行 `AutoRun.exe`

### 基本使用

#### 1. 新增排程
1. 點擊「新增」按鈕
2. 填寫排程資訊：
   - **名稱**：排程的名稱（例如：開啟工作軟體）
   - **程式**：要執行的程式路徑（使用「...」按鈕選擇）
   - **參數**：命令列參數（選填）
   - **時間**：執行時間（HH:mm 格式）
   - **週期**：選擇執行的星期
3. 點擊「加入/更新」儲存

#### 2. 啟動排程器
1. 點擊「啟動排程器」按鈕
2. 排程器會在背景執行，定時檢查並執行排程

#### 3. 最小化到托盤（選用）
1. 勾選「最小化至系統夾」
2. 點擊「啟動排程器」後會自動最小化到系統托盤
3. 或在排程器運行中勾選，會立即最小化

#### 4. 從托盤恢復
- 雙擊托盤圖示，或
- 右鍵托盤圖示 → 選擇「開啟視窗」

## 📖 使用範例

### 範例 1：每天早上開啟工作軟體
```
名稱：開啟 VS Code
程式：C:\Program Files\Microsoft VS Code\Code.exe
參數：D:\Projects\MyProject
時間：09:00
週期：每天
啟用：✓
```

### 範例 2：週末執行備份
```
名稱：系統備份
程式：D:\Scripts\backup.bat
參數：
時間：02:00
週期：六、日
啟用：✓
```

### 範例 3：工作日定時提醒
```
名稱：休息提醒
程式：C:\Tools\reminder.exe
參數：--message "該休息了"
時間：10:30
週期：一、二、三、四、五
啟用：✓
```

## 🎯 功能說明

### 三個獨立功能

| 功能 | 說明 | 控制方式 |
|------|------|----------|
| **最小化至系統夾** | 將視窗隱藏到系統托盤 | 勾選 checkbox |
| **啟動排程器** | 控制排程的執行 | 點擊按鈕 |
| **開機自動啟動** | Windows 開機時自動啟動程式 | 勾選 checkbox |

### 智能聯動

- ✅ 勾選「最小化至系統夾」+ 點擊「啟動排程器」→ 自動最小化
- ✅ 排程器運行中 + 勾選「最小化至系統夾」→ 立即最小化
- ✅ 從托盤恢復視窗 → 保持原有設定狀態

### 系統托盤功能

**右鍵選單**：
- 開啟視窗
- 啟動排程器 / 停止排程器
- 退出

**滑鼠停留提示**：
```
排程器: 執行中
共 3 個排程 (2 個啟用)
```

## 📁 設定檔案

程式會在執行目錄建立兩個 JSON 設定檔：

- **setting.json** - 排程項目資料
- **settings.json** - 應用程式設定（最小化選項、開機啟動）

## ⚙️ 技術資訊

- **開發框架**：.NET 8.0
- **UI 框架**：Windows Forms
- **程式語言**：C# 12.0
- **資料格式**：JSON

### 排程精度
- 檢查間隔：每 5 秒
- 時間精度：分鐘級別（HH:mm）
- 防重複機制：同一分鐘內不會重複執行

## 🐛 已知限制

- ⚠️ 排程時間精度為分鐘級別，不支援秒級精度
- ⚠️ 程式啟動可能受到系統權限限制
- ⚠️ 開機自動啟動僅支援當前用戶
- ⚠️ 執行紀錄不會保存到檔案
